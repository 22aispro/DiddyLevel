DiddyLevel

DiddyLevel is a server-side leveling framework for Roblox that handles XP, player progression, and customizable leveling logic.

It is designed to be simple, flexible, and extensible for any game type.

Installation

  1. Place the DiddyLevel folder in ServerStorage
  2. Require the module from a server script

local DiddyLevel = require(game.ServerStorage.DiddyLevel)

Core Concepts

DiddyLevel is built from three main modules:

  • DiddyLevel – public API
  • LevelHandler – level & player state
  • LevelMethods – custom XP triggers

Levels

A level is defined using the following structure:


export type Level = {
  LevelRank: number,
  RequiredXP: number,
}

Levels are registered using InitializeLevels. 


DiddyLevel.InitializeLevels({
  { LevelRank = 1, RequiredXP = 0 },
  { LevelRank = 2, RequiredXP = 100 },
  { LevelRank = 3, RequiredXP = 250 },
})

Levels are indexed internally by LevelRank. Duplicate ranks or XP values are ignored.

Players

Each player is tracked using a PlayerLevelObject. 


export type PlayerLevelObject = {
  Linked: Player,
  Current: Level,
  CurrentXP: number,
  NextLevel: Level?,
}

Players are initialized when they join:


DiddyLevel.InitializePlayer(player)

If no data is provided, the player starts at level 1 automatically.

Awarding XP

XP is awarded using AwardXP. Level-ups are handled automatically. 


DiddyLevel.AwardXP(player, 50)

If enough XP is earned, the player may level up multiple times in a single award.

Leveling Methods

Leveling methods allow you to define custom XP logic such as kills, quests, or actions. 


export type LevelingMethod = {
  key: string,
  xp: number,
  callback: () -> any,
}

Creating a Method


DiddyLevel.CreateLevelMethod({
  key = "KillEnemy",
  callback = function(player, character, enemy)
    DiddyLevel.AwardXP(player, 25)
  end,
})

Forcing a Method


DiddyLevel.ForceMethod("KillEnemy", player, character, enemy)

Example Setup


local Players = game:GetService("Players")
local DiddyLevel = require(game.ServerStorage.DiddyLevel)

DiddyLevel.InitializeLevels({
  { LevelRank = 1, RequiredXP = 0 },
  { LevelRank = 2, RequiredXP = 100 },
})

Players.PlayerAdded:Connect(function(player)
  DiddyLevel.InitializePlayer(player)
end)
