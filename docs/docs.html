DiddyLevel

Introduction Installation Usage API

Introduction

DiddyLevel is a flexible XP and leveling framework designed for Roblox server-side systems.

Installation

  1. Download the repository
  2. Place DiddyLevel in ServerStorage
  3. Require it in your server scripts

Basic Usage


local DiddyLevel = require(path.to.DiddyLevel)

DiddyLevel.InitializePlayer(player)
DiddyLevel.AddXP(player, 100)

API

InitializePlayer(player)

Registers a new player with default XP and level.
