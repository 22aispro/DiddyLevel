API Reference

Complete reference for all DiddyLevel functions.

InitializeLevels(levelList)

Registers all levels for your game.


DiddyLevel.InitializeLevels({
  { LevelRank = 1, RequiredXP = 0 },
  { LevelRank = 2, RequiredXP = 100 },
})

InitializePlayer(player, data?)

Registers a player with optional starting data.


DiddyLevel.InitializePlayer(player, optionalData)

AwardXP(player, xp)

Awards XP and handles automatic level-ups.


DiddyLevel.AwardXP(player, 25)

CreateLevelMethod(methodData)

Creates a custom leveling method with a key and callback.


DiddyLevel.CreateLevelMethod({
  key = "KillEnemy",
  callback = function(player, char)
    DiddyLevel.AwardXP(player, 10)
  end
})

ForceMethod(key, player, char, extraArgs)

Manually executes a registered method by its key.


DiddyLevel.ForceMethod("KillEnemy", player, character, extra_args)
