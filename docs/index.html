DiddyLevel

DiddyLevel is a server-side XP and leveling framework for Roblox. It provides a simple API for defining levels, tracking player progress, and awarding XP through customizable methods.

Getting Started

  1. Define your levels
  2. Initialize players on join
  3. Award XP through gameplay

local DiddyLevel = require(game.ServerStorage.DiddyLevel)

DiddyLevel.InitializeLevels({
  { LevelRank = 1, RequiredXP = 0 },
  { LevelRank = 2, RequiredXP = 100 },
})
