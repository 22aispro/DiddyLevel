DiddyLevel

A server-side XP and leveling framework for Roblox. Built to be simple, extensible, and Studio-friendly.

Installation


local DiddyLevel = require(game.ServerStorage.DiddyLevel)

Defining Levels


DiddyLevel.InitializeLevels({
  { LevelRank = 1, RequiredXP = 0 },
  { LevelRank = 2, RequiredXP = 100 },
})

Awarding XP


DiddyLevel.AwardXP(player, 25)

Leveling Methods


DiddyLevel.CreateLevelMethod({
  key = "Kill",
  callback = function(player, character)
    DiddyLevel.AwardXP(player, 10)
  end
})
