DiddyLevel

A flexible XP & leveling framework for Roblox games. Simple API, fully server-side, and Studio-friendly.

Get Started API Reference 

-- Example usage
local DiddyLevel = require(game.ServerStorage.DiddyLevel)

DiddyLevel.InitializeLevels({
  { LevelRank = 1, RequiredXP = 0 },
  { LevelRank = 2, RequiredXP = 100 },
})

DiddyLevel.AwardXP(player, 50)
