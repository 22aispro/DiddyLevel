DiddyLevel

DiddyLevel is a server-side XP and leveling framework for Roblox. It provides a structured way to define levels, track player progression, and award experience through customizable methods.

Getting Started

To start using DiddyLevel, define your levels and initialize players when they join the game. 


local DiddyLevel = require(game.ServerStorage.DiddyLevel)

DiddyLevel.InitializeLevels({
  { LevelRank = 1, RequiredXP = 0 },
  { LevelRank = 2, RequiredXP = 100 },
})

For a complete walkthrough, see the full documentation.

API Summary

  • InitializeLevels(levels)
  • InitializePlayer(player, data?)
  • AwardXP(player, xp)
  • CreateLevelMethod(method)
  • ForceMethod(key, player, character, args)
