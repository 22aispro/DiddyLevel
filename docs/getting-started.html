DiddyLevel — Getting Started

Learn how to install and initialize DiddyLevel in your Roblox game.

Installation

Place the DiddyLevel folder into ServerStorage and require it in a server script:


local DiddyLevel = require(game.ServerStorage.DiddyLevel)

Defining Levels

Define the levels that players can reach:


DiddyLevel.InitializeLevels({
  { LevelRank = 1, RequiredXP = 0 },
  { LevelRank = 2, RequiredXP = 100 },
  { LevelRank = 3, RequiredXP = 250 },
})

Initialize Players

Register players when they join:


local Players = game:GetService("Players")

Players.PlayerAdded:Connect(function(player)
  DiddyLevel.InitializePlayer(player)
end)

Award XP

Grant XP to players:


DiddyLevel.AwardXP(player, 50)
