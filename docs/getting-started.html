Getting Started

Install and initialize DiddyLevel in your Roblox game.

Installation


local DiddyLevel = require(game.ServerStorage.DiddyLevel)

Define Levels


DiddyLevel.InitializeLevels({
  1;
  2;
  3;
})

Initialize Players


local Players = game:GetService("Players")

Players.PlayerAdded:Connect(function(player)
  DiddyLevel.InitializePlayer(player)
end)

Award XP


DiddyLevel.AwardXP(player, 50)
